# Veloura

A full-stack web application built with React, Node.js, and Prisma.

## 🚀 Tech Stack

- **Frontend**: React 19 + Vite + Tailwind CSS
- **Backend**: Node.js + Express + TypeScript
- **Database**: Prisma ORM
- **Package Manager**: pnpm
